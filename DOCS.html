📁 Website Files Included

index.html
Main website page. Contains all sections: hero, builds, forms, photography, map, and footer. Open this in browser to preview.
styles.css
All website styling, colors, layout, and responsive design. Edit this to change colors/fonts/spacing.
script.js
JavaScript for functionality: forms, mobile menu, Google Maps, animations, and interactions.
netlify.toml
Netlify configuration file. Leave as-is - tells Netlify how to deploy your site.
package.json
Project metadata. Shows this is your project and can hold future dependencies.
